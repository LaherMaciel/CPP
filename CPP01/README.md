# CPP Module 01 - Memory Allocation & References

## Overview

Module 01 covers memory management, references, pointers to member functions, and the switch statement. Builds on the OOP fundamentals from Module 00.

## Learning objectives

- Dynamic memory allocation with `new` and `delete`
- Array allocation with `new[]` and `delete[]`
- References and their differences from pointers
- Pointers to member functions
- Switch statement usage

## Exercises

| Exercise | Name | Key concepts |
|----------|------|-------------|
| ex00 | BraiiiiiiinnnzzzZ | Stack vs heap allocation, `new`/`delete`, object lifetime |
| ex01 | Moar brainz! | `new[]`, `delete[]`, allocating arrays of objects |
| ex02 | HI THIS IS BRAIN | References vs pointers, address comparison |
| ex03 | Unnecessary violence | References as class members, when to use ref vs pointer |
| ex04 | Sed is for losers | `std::ifstream`, `std::ofstream`, string replacement without `std::string::replace` |
| ex05 | Harl 2.0 | Pointers to member functions, avoiding if/else chains |
| ex06 | Harl filter | `switch`/`case`, fall-through behavior |

## Compilation

```bash
cd ex00  # or ex01–ex06
make
```

Compiles with `-Wall -Wextra -Werror -std=c++98`.
