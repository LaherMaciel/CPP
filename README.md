## 🛠️ Setup and Usage

### Prerequisites
- C++ compiler (g++, clang++, or MSVC)
- Make (optional, for build automation)

### Compilation
```bash
# Compile a specific project
cd CPP_Module_00
g++ -Wall -Wextra -Werror -std=c++98 *.cpp -o program_name

# Or use make if Makefile is provided
make
```

### Running
```bash
./program_name
```
